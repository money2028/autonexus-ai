# AutoNexus AI 工作流编辑器

## 概述
可视化拖拽界面，用于定义 AI 代理的行为流程。

## 功能
- **节点库**：触发器、回复消息、条件分支、API调用、延迟等待、结束节点。
- **拖拽编辑**：从左侧拖拽节点到画布，自由布局。
- **属性配置**：点击节点，在右侧面板编辑详细参数。
- **节点连接**：通过输入/输出点连接节点（连线功能为示意，待完善）。
- **导出JSON**：将工作流导出为结构化配置，供代理引擎执行。

## 使用步骤
1. 打开 `workflow-editor.html` 于浏览器。
2. 从左侧拖拽所需节点到画布。
3. 点击节点，在右侧配置属性（如回复内容、API地址等）。
4. 使用导出按钮获取 JSON 配置。

## 导出的工作流 JSON 示例
```json
{
  "name": "客服接待流程",
  "nodes": [
    { "id": "node_1", "type": "trigger", "event": "message.received" },
    { "id": "node_2", "type": "reply", "text": "您好，请问有什么可以帮您？" },
    { "id": "node_3", "type": "condition", "field": "content", "operator": "contains", "value": "价格" },
    { "id": "node_4", "type": "reply", "text": "我们的标准套餐起价为每月 $299。" }
  ],
  "connections": [
    { "from": "node_1", "to": "node_2" },
    { "from": "node_2", "to": "node_3" }
  ]
}
```

## 后续开发
- **后端集成**：保存/加载工作流到数据库。
- **连线逻辑**：实现完整的节点连接与流程验证。
- **实时预览**：模拟运行工作流，测试效果。
- **节点扩展**：更多节点类型（数据库查询、变量赋值、人工接管等）。

## 技术栈
- 纯 HTML/CSS/JavaScript（无依赖，可离线运行）。
- 可轻松嵌入到现有管理后台。
